# Archmule-aut-wdio
Project to training about automation with Mocha and Chai frameworks
Using the following technologies:
```
- Java Script
- Mocha
- Chai
- Webdriver IO
- ESLint
```
First download the dependencies with
```
npm install
```
To run the tests execute:
```
npm test
```
